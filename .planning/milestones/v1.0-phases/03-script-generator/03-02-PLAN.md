**Functions:**

1. `show_module_menu() -> dict | None`:
   - Call `list_modules()` to get modules.
   - Create `Table(title="Modules")` with columns: `#` (bold), `Module`.
   - Add row for each module: `str(i)`, `mod["title"]` where i starts at 1.
   - Add row `"0"`, `"Quit"`.
   - `console.print(table)`
   - `choice = IntPrompt.ask("Select module", choices=[str(i) for i in range(len(modules) + 1)], default=0)`
   - If choice == 0, return None.
   - Return `modules[choice - 1]`

2. `show_lesson_menu(module: dict) -> dict | None`:
   - Call `list_lessons_for_module(module["num"])` to get lessons.
   - Create `Table(title=f"Module {module['num']}: {module['title']}")` with columns: `#` (bold), `ID`, `Title`, `Status`.
   - For each lesson: add row with index, `les["id"]`, `les["title"]`, `Text(les["status"], style=STATUS_STYLES.get(les["status"], "white"))`.
   - Add row `"0"`, `""`, `"Back to modules"`, `""`.
   - `console.print(table)`
   - `choice = IntPrompt.ask("Select lesson", choices=[str(i) for i in range(len(lessons) + 1)], default=0)`
   - If choice == 0, return None.
   - Return `lessons[choice - 1]`

3. `display_script(script_text: str, lesson_title: str)`:
   - `syntax = Syntax(script_text, "markdown", theme="monokai", word_wrap=True)`
   - `console.print(Panel(syntax, title=lesson_title, border_style="cyan"))`
   - Word count: `wc = len(script_text.split())`
   - `console.print(f"  Word count: {wc}")`

4. `open_in_editor(content: str) -> str`:
   - `editor = os.environ.get("EDITOR", "notepad")`
   - `editor_parts = editor.split()`
   - Create temp file: `tempfile.NamedTemporaryFile(suffix=".md", mode="w", delete=False, encoding="utf-8")`
   - Write content, get tmp_path.
   - `subprocess.run(editor_parts + [tmp_path], check=False)` (check=False per D-12 research — editors may return non-zero)
   - Read back: `Path(tmp_path).read_text(encoding="utf-8")`
   - Delete temp file: `Path(tmp_path).unlink(missing_ok=True)` in finally block.
   - Return the read-back text.

5. `review_script(lesson_id: str, lesson_title: str, script_text: str, bootcamp_excerpts: str = "", lesson_outline: str = "") -> bool`:
   - This is the (a)ccept/(e)dit/(r)egenerate/(s)kip loop.
   - Display script with `display_script(script_text, lesson_title)`.
   - Loop:
     - `choice = Prompt.ask("(a)ccept / (e)dit / (r)egenerate / (s)kip", choices=["a", "e", "r", "s"], default="s")`
     - If `"a"`: call `path = save_script(lesson_id, lesson_title, script_text)`. Print `f"Saved: {path}"`. Return `True`.
     - If `"s"`: print "Skipped." Return `False`.
     - If `"e"`: call `edited = open_in_editor(script_text)`. Display edited script. Ask `Prompt.ask("(a)ccept edited / (d)iscard", choices=["a", "d"], default="d")`. If "a": `save_script(...)` + return True. If "d": continue loop (re-display original, re-ask).
     - If `"r"`: `note = Prompt.ask("Regenerate with note")`. Call `script_text = generate_script(lesson_outline, bootcamp_excerpts, revision_note=note)`. Re-display with `display_script(...)`. Continue loop.

Note on `review_script` signature: `bootcamp_excerpts` and `lesson_outline` are needed for the regenerate path. The caller (generate.py) passes them from assemble_context output.

Ensure tests pass: `test_skip_does_not_save` mocks `Prompt.ask` returning "s" and checks `save_script` not called. `test_accept_saves_script` mocks `Prompt.ask` returning "a" and checks `save_script` called. Adjust function signature to match what the tests expect — the tests pass `(lesson_id, lesson_title, script_text)` as positional args.

**Main function — `def main()`:**

The flow per D-06/D-07/D-10/D-15:
1. Outer loop (module selector):
   - `module = show_module_menu()` — returns None if user selects Quit (0).
   - If None: `console.print("Goodbye!")` and break.
2. Inner loop (lesson selector):
   - `lesson = show_lesson_menu(module)` — returns None if user selects Back (0).
   - If None: break inner loop (goes back to module selector).
3. Generate and review:
   - Print `f"Generating script for {lesson['id']}: {lesson['title']}..."`
   - `ctx = assemble_context(lesson["lesson_num"], ...)` — NOTE: `assemble_context` takes `(module_num, lesson_num)` where `module_num` is the module's `num` field from the module dict. The lesson dict from `list_lessons_for_module` has `lesson_num`. So: `ctx = assemble_context(module["num"], lesson["lesson_num"])`.
   - Wait — check: `list_lessons_for_module` returns dicts with key `lesson_num`. And `assemble_context(module_num, lesson_num)` matches `get_lesson(module_num, lesson_num)`. So: `ctx = assemble_context(module["num"], lesson["lesson_num"])`.
   - `script = generate_script(ctx["lesson_outline"], ctx["bootcamp_excerpts"])`
   - `review_script(ctx["lesson_id"], ctx["lesson_title"], script, bootcamp_excerpts=ctx["bootcamp_excerpts"], lesson_outline=ctx["lesson_outline"])`
   - After review returns (whether accepted or skipped): continue inner loop (back to lesson list per D-15 which says "return to module selector" but the flow is: after accept, go back to lesson selector which allows continuing or going back to modules).

Actually, re-reading D-15: "After accept: immediately return to module selector (continuous flow)." So after accept (review_script returns True), break inner loop to go back to module selector. After skip/other (returns False), stay in inner loop (lesson selector).

Correction to flow:
- `accepted = review_script(...)`
- If `accepted`: break inner loop (back to module selector per D-15)
- If not accepted: continue inner loop (stay in lesson selector)

4. Error handling:
   - Wrap generate+review in try/except:
     - `ValueError` (missing API key, lesson not found): `console.print(f"[red]Error: {e}[/red]")`, continue inner loop.
     - `KeyboardInterrupt`: graceful exit.
     - General `Exception`: `console.print(f"[red]Unexpected error: {e}[/red]")`, continue.
